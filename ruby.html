Everything is an object.

What is not an object, is a method call. 

puts "Hello World!"
1 + 2

Everything has a value.

def method
  if nil
    'nil'
  else
    'result'
  end
end
@result = method

Our friends:

[
  [1, 2, 3],
  1..3,
  %w(a b c),
  '%5s' % 'pad',
  {
    :user => 'Timmy',
    :ability => :"ta-da!"
  }
].each do |item|
  puts item.class, item.inspect
end
