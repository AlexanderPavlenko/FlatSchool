Everything is an object.

What is not an object, is a method call. 

puts "Hello World!"
1 + 2
> nil.class
=> NilClass 
> nil.class.superclass
=> Object

Flow is under control.

if value
  'anything'
elsif value.nil?
  'nil'
else
  'false'
end
for i in 1.upto(5)
  next if i == 1
  break if i == 5
  puts i
end

Flow is under control.

case value

when 'red'
  :bad

  :fault

unless value  # equals to: if !value

Everything has a value.

  if true
    true && 'result'
  else
    'this'
  end
end
result = method
